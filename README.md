# Firebase Blog for React.js

Blog component powered by Firebase realtime database

## Install

```sh
npm i react-firebase-blog
```


## Use


```jsx
import FireBlog from 'react-firebase-blog';

const config = {

	// Firebase Config

}

render() {
  return (
    <FireBlog config={config} />   
  );
}
```
